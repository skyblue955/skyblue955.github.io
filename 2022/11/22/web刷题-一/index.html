正则表达式

​ 循环匹配的—- ( ?R )

​ ( ?R ) ：匹配一次 

 ( ?R )+   ：匹配一次或多次

​ ( ?R )* ：匹配0次或多次

​ ( ?R )？ ：匹配0次或1词

​

​ 因为是递归循环匹配的（相当于一直在循环匹配），前两个如果匹配到的话是没有退出条件的（所以匹配不成功），而后两个是由退出条件（即为0次，时），所以可以匹配成功

​ 例子：

​ a(b(c()))—–若按第二个匹配来的话，那么最里的括号里应该还有能匹配的

​ 按第三个来的话，匹配到 ***c()***时，此时没有能匹配的了，匹配次数为0.相 当于在此停止匹配了
